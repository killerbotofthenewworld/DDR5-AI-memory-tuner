## 🚀 One-Click Setup

### Linux/macOS
```bash
git clone https://github.com/your-repo/ddr5-ai-sandbox-simulator.git
cd ddr5-ai-sandbox-simulator
chmod +x setup.sh
./setup.sh
```

### Windows
```cmd
git clone https://github.com/your-repo/ddr5-ai-sandbox-simulator.git
cd ddr5-ai-sandbox-simulator
setup.bat
```

Then run: `./run_ddr5_simulator.sh` (Linux/macOS) or `run_ddr5_simulator.bat` (Windows)
